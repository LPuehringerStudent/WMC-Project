Data Types

In programming, data types specify what kind of data can be stored in a variable...

int age = 25;
float temp = 36.5f;
char grade = 'A';

Input/Output

Input/Output (I/O) is the process of reading input and displaying output...

Console.Write("Enter a number: ");
int num = int.Parse(Console.ReadLine());

Mathematics

Math in programming includes addition, subtraction, etc...

int a = 10, b = 5;
Console.WriteLine(a + b);

Arrays

Arrays store multiple values in a single variable...

int[] numbers = {1, 2, 3};

Enums

Enums define a set of named constants...

enum Week {Mon, Tue, Wed};

Records

Records are groupings of related variables...

struct Student {
string Name;
int Age;
}
