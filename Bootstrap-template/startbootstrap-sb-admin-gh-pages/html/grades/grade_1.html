Data Types

Data types determine the kind of data a variable can hold. Common types include:

  • int: Whole numbers (e.g., 1, -5, 42)
  • decimal: Decimal numbers (e.g., 3.14, 0.5)
  • char: A single character (e.g., 'A', 'z')
  • bool: Boolean values (true or false)
int age = 25;
float temp = 36.5f;
char grade = 'A';
bool isStudent = true;

Exercise

Try declaring variables of each type and print their values using Console.WriteLine().

Input/Output

Input/Output (I/O) allows you to interact with the user. You can get input using Console.ReadLine() and display output using Console.WriteLine().

Console.Write("Enter your name: ");
string name = Console.ReadLine();
Console.WriteLine("Hello, " + name + "!");

Exercise

Ask the user to enter two numbers and print their sum.

Console.Write("Enter first number: ");
int a = int.Parse(Console.ReadLine());
Console.Write("Enter second number: ");
int b = int.Parse(Console.ReadLine());
Console.WriteLine("Sum: " + (a + b));

Mathematics

You can perform basic math operations using +, -, *, /, and % (modulo).

int x = 12;
int y = 5;

Console.WriteLine(x + y); // 17
Console.WriteLine(x % y); // 2

Console.WriteLine("Subtract: " + (x - y));
Console.WriteLine("Multiply: " + (x * y));
Console.WriteLine("Divide: " + (x / y));
Console.WriteLine("Modulo: " + (x % y));

Arrays

Arrays are used to store multiple values of the same data type in one variable. Indexes start at 0.

Example: Basic Array

int[] numbers = {10, 20, 30, 40};
Console.WriteLine(numbers[1]); // 20

Exercise

Print the first and last values of an array. Then change one value and print the array using a loop.

Looping Through Arrays

foreach (int n in numbers)
{
    Console.WriteLine(n);
}

2D Arrays

2D arrays look like tables. You access values using two indices: row and column.

int[,] grid = {
    {1, 2, 3},
    {4, 5, 6}
};

Console.WriteLine(grid[1, 2]); // Outputs 6

Array of Arrays (Jagged Arrays)

Jagged arrays are arrays of arrays. Each row can be a different length.

int[][] jagged = new int[2][];
jagged[0] = new int[] {1, 2, 3};
jagged[1] = new int[] {4};

Console.WriteLine(jagged[0][2]); // 3

Exercise

Write nested loops to print a jagged array.

for (int i = 0; i < jagged.Length; i++)
{
    for (int j = 0; j < jagged[i].Length; j++)
    {
        Console.Write(jagged[i][j] + " ");
    }
    Console.WriteLine();
}

Enums

Enums are a way to name a set of related constant values. They make code more readable.

enum Weekday { Monday, Tuesday, Wednesday, Thursday, Friday };

Weekday today = Weekday.Tuesday;
Console.WriteLine(today); // Tuesday

Exercise

Try printing each day of the week using a loop.

foreach (Weekday day in Enum.GetValues(typeof(Weekday)))
{
    Console.WriteLine(day);
}

Records

Records group related variables into one unit. They are useful for representing objects like students, books, etc.

Example: Struct

Record Student
{
    public string Name;
    public int Age;
}

Student s;
s.Name = "Lisa";
s.Age = 17;

Console.WriteLine(s.Name + " is " + s.Age + " years old.");
